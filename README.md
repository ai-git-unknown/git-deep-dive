# How Git Works — A Deep Dive

A self-contained, single-file HTML explainer of Git's internals, built step by step:

- The three states (working dir / index / repo)
- Inside `.git/`
- The content-addressable object database (blob, tree, commit, tag)
- The index / staging area
- References & HEAD
- Branching, merging (fast-forward + 3-way), rebase
- Remotes, packfiles & GC, reflog
- A plumbing cheat sheet

Open `index.html` in any browser — no dependencies, no network.

Published at: https://ai-git-unknown.github.io/git-deep-dive/
